Aplikasi CRUD Data Buku

Aplikasi CRUD Data Buku merupakan sebuah aplikasi berbasis web yang digunakan untuk mengelola data buku. Aplikasi ini dibuat menggunakan bahasa pemrograman **PHP** sebagai backend, **MySQL** sebagai basis data, serta **HTML, CSS, dan Bootstrap** sebagai frontend untuk tampilan antarmuka pengguna.

Aplikasi ini menerapkan konsep **CRUD (Create, Read, Update, Delete)** yang memungkinkan pengguna untuk:
* Menambahkan data buku baru
* Menampilkan daftar buku yang tersedia
* Mengubah (edit) data buku yang sudah ada
* Menghapus data buku beserta gambar dari sistem
Setiap data buku terdiri dari **judul buku, pengarang, penerbit, kategori, dan gambar**. Aplikasi ini juga mendukung fitur **upload gambar**, di mana pengguna dapat menambahkan atau mengganti gambar buku saat proses edit tanpa menghapus gambar lama jika tidak diperlukan.

Struktur aplikasi dipisahkan berdasarkan fungsinya, di mana:
* **index.php** digunakan untuk menampilkan daftar data buku
* **olah.php** digunakan untuk form tambah dan edit data buku
* **proses.php** digunakan untuk menangani proses penyimpanan, pengubahan, dan penghapusan data
* **koneksi.php** digunakan untuk mengatur koneksi ke database

Aplikasi ini dirancang dengan tampilan yang sederhana, rapi, dan responsif menggunakan Bootstrap serta CSS terpisah untuk mempercantik antarmuka. Dengan adanya aplikasi ini, pengelolaan data buku menjadi lebih **terstruktur, efektif, dan mudah digunakan**, terutama untuk kebutuhan pembelajaran CRUD berbasis PHP dan MySQL.
